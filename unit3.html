Geoid, Ellipsoid aur Datum kya hain? Datum Transformation explain karo.
▼
  • Geoid — Earth ka actual irregular shape (mean sea level surface). Gravity se influenced. Height measurement ke liye.
  • Ellipsoid — Earth ka smooth mathematical model (slightly flattened sphere). GPS ke liye WGS84 standard hai.
  • Datum — Ellipsoid + uski Earth ke relative position = coordinate system ka foundation
Geoid = Bumpy actual Earth. Ellipsoid = Smooth ball (for calculations). WGS84 = GPS wala datum.

Transformation Types:

  • Coordinate Transformation — Same datum pe alag systems (Cartesian ↔ Geographic)
  • Datum Transformation — Alag datums ke beech shift (India's old Everest Datum → WGS84)
GPS & Satellite Positioning
🛰️
Apr'19 Q3b · Apr'23 Q3d
GPS kya hai? Kaise kaam karta hai? 🔥 Hot
▼
GPSUS Government ka satellite system jo kisi bhi location ki Latitude, Longitude, Altitude precise tarike se deta hai.

3 Segments:

  • Space Segment — 24+ satellites Earth orbit mein (6 planes × 4 satellites)
  • Control Segment — Ground stations jo satellites monitor karte hain
  • User Segment — Tumhara GPS phone/device

Working — Trilateration:

  • Receiver ≥ 3 satellites se signals receive karta hai
  • Signal travel time se distance = Speed of Light × Time
  • 3 circles ka intersection = 2D position
  • 4th satellite = Altitude + Error correction
3 satellites se distances: 200km, 250km, 180km → teen circles ka intersection = tumhari exact location!

GPS Errors (Space Segment):

  • Satellite Clock Error — Atomic clock ki minute inaccuracy
  • Orbital Error — Satellite predicted orbit se thoda alag
  • Atmospheric Error — Ionosphere/Troposphere mein signal delay
  • Multipath — Signal buildings se reflect hokar aata hai
GLONASS (Russia) = GPS ka Russian equivalent. 24 satellites. GPS+GLONASS receiver zyada accurate hota hai.

Apr'19 Q3d · Apr'25 Q3e
Trend Surface Fitting kya hai?
▼
Trend Surface FittingScattered data points pe mathematical polynomial surface fit karna jo overall data trend represent kare.
Formula (1st order/linear): Z = a₀ + a₁X + a₂Y Example — Rainfall Stations: Station | X | Y | Rainfall(mm) A | 0 | 0 | 120 B | 5 | 0 | 110 C | 0 | 5 | 130 Trend: West→East mein rainfall decrease ho raha hai → Unknown point (3,3) ka rainfall ≈ 118mm estimate kar sakte hain
  • Higher order polynomial = more complex surface
  • Use: Groundwater level, temperature distribution prediction

Apr'19 Q3f · Apr'23 Q3f · Nov'25 Q3f
Kriging kya hai? Interpolation explain karo. 📌 Med
▼
InterpolationKnown data points se unknown locations ki values estimate karna — continuous surfaces banana.
KrigingGeostatistical method — nearby points ko weighted average se use karta hai. Closer = more weight.
5 rain gauges: A=120, B=115, C=108, D=125, E=112 mm. Unknown point P ke nearest gauges A(2km) aur D(3km) → Kriging estimate: P ≈ 121mm.

Other Methods:

  • IDW — Inverse Distance Weighting: Closer = more weight. Simple but less accurate than Kriging.
  • TIN Triangulation — Data points se triangles → surface. Terrain ke liye popular.
  • Spline — Smooth curve fit — minimum curvature surface.

RMS (Root Mean Square) ErrorMeasured vs Actual positions ka average error. Map accuracy assess karne ke liye use hota hai.
RMS = √[ Σ(error²) / n ] Control Actual Measured Error Error² Point X X P1 100 101.2 1.2 1.44 P2 200 198.8 -1.2 1.44 P3 300 301.5 1.5 2.25 P4 400 399.0 -1.0 1.00 Sum = 6.13 RMS = √(6.13 / 4) = √1.53 = 1.24 meters → Positions on average 1.24m galat hain.

Satellite-based positioning kya hai?
▼

GNSS (Global Navigation Satellite System) systems:

  • GPS (USA) — 24+ satellites, WGS84 datum
  • GLONASS (Russia) — 24 satellites, PZ-90 datum
  • Galileo (Europe) — 30 satellites (civilian focus)
  • NavIC (India) — 7 satellites, India region focus

Issues in combining data from multiple sources (Nov'19 Q3e):

  • Scale differences — 1:25000 vs 1:50000 maps
  • Different datums — Everest vs WGS84
  • Different formats — Shapefile vs GeoJSON
  • Temporal mismatch — 2010 data vs 2023 data
