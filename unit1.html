GIS Basics
🌍
Apr'19 Q1a · Apr'23 Q2a · Nov'25 Q1f
GIS kya hai? Iske capabilities kya hain? 🔥 Hot
▼
DefinitionGIS (Geographic Information System) = Computer system jo location-based data ko collect, store, analyze aur display karta hai.

4 Capabilities:

  • Data Capture — GPS, satellite se data lena
  • Data Storage — Database mein save karna
  • Data Analysis — Patterns dhundhna, queries run karna
  • Data Visualization — Maps aur charts mein dikhana
Google Maps tumhara location leta hai → route analyze karta hai → screen pe dikhata hai. Yahi GIS ka kaam hai!

Models & Geographic Phenomena
🧩
Apr'19 Q1c · Apr'23 Q1c · Apr'25 Q1c · Nov'25 Q1e
Model kya hai? GIS mein modelling kyun zaroori hai? 🔥 Hot
▼
ModelReal world ka simplified representation — taaki analyze aur samjha ja sake.

3 Types:

  • Conceptual — Idea/design: kaunsa data chahiye?
  • Logical — Structure: data kaise organize hoga?
  • Physical — Actual database on computer

Kyun useful hai?

  • Real world complex hai — model simple banata hai
  • Bina field experiment ke analysis possible
  • Future predictions possible (2030 mein flood zone kahan?)
Google Maps pe roads = model. Real world mein hazaaron details hain, map sirf zaroori cheezein dikhata hai.
📍
Apr'19 Q1d · Nov'19 Q1d · Apr'25 Q1d · Nov'25 Q1b
Geographic Field vs Geographic Object. Object ke 4 parameters? 🔥 Hot
▼
Geographic FieldContinuously exist karne wali phenomenon — har location pe ek value. E.g. Temperature, Rainfall.
Geographic ObjectDiscrete (alag) entity with specific location. E.g. Roads, Buildings, Cities.

4 Parameters of Geographic Object:

  • Position — Kahan hai? (Lat/Long)
  • Shape — Kaisa dikhta hai? (Point/Line/Polygon)
  • Size — Kitna bada? (Area, Length)
  • Attributes — Extra info (Name, Population, Type)
Mumbai: Position=19.07°N 72.87°E, Shape=Polygon, Size=603 km², Attributes: Population 2 crore.

Field ke Data Types:

  • Nominal — Categories, no order (Forest/Water/Urban)
  • Ordinal — Ordered categories (Poor/Good/Excellent)
  • Interval — Numbers, no true zero (Temperature °C)
  • Ratio — Numbers with true zero (Distance, Weight)

Mathematical properties of geometric space kya hain?
▼
  • Dimension — 0D=Point, 1D=Line, 2D=Area, 3D=Volume
  • Continuity — Space mein koi gaps nahi hote

Topology & Spatial Relationships
🔗
Apr'19 Q1e
Topology aur spatial relationships kya hain?
▼
TopologyFeatures ke beech spatial relationships define karta hai — Shape/Size change hone pe bhi relationships sahi rehti hain.

Key Relationships:

  • Connectivity — Roads junction pe milti hain
  • Adjacency — Maharashtra-Karnataka border share karte hain
  • Containment — City is inside a State
  • Overlap — Features partially over each other
  • Routing/Navigation ke liye zaroori
  • Data errors (gaps/overlaps) detect karna
  • Network analysis (pipe/road networks)
Temporal Dimension
⏰
Apr'19 Q1f · Apr'23 Q1f · Apr'25 Q1f · Nov'19 Q1c · Nov'25 Q1c
Temporal dimension kya hai? Time in GIS kaise represent karte hain? 🔥 Hot
▼
Temporal DimensionGIS mein location (where) + attributes (what) + time (when) — data time ke saath kaise change hota hai.

Time ke Types:

  • Time Instant — Ek specific moment: "15 Aug 2023 flood aaya"
  • Time Interval — Ek period: "June-Sept monsoon season"
  • Valid Time — Real world mein kab hua (building 2005 mein bani)
  • Transaction Time — Database mein kab enter hua

Spatiotemporal Models:

  • Snapshot Model — Har time period ka alag map store karo
  • Space-Time Composite — Sirf changes store karo (efficient)
Mumbai land use 1990 → 2000 → 2010 → 2020 — dekh sakte hain slums kahan se urban area ban gayi. Yahi temporal GIS hai!
Apr'25 Q1b mein El Niño aaya — yeh climate event hai jo Pacific Ocean temperature warm karta hai. GIS mein multiple years ke temperature maps compare karte hain.
Tessellations & Quad Tree
🔲
Nov'19 Q1e-f · Apr'23 Q1d · Nov'25 Q1a
Regular vs Irregular Tessellations. Quad Tree banao. 📌 Med
▼
TessellationSpace ko tiles mein divide karna bina gaps aur overlaps ke. Raster data ka base hai.
  • Regular — Sab tiles same size/shape: squares (pixels/grid), triangles, hexagons
  • Irregular — Tiles different size/shape: TIN (terrain ke liye)
Regular = Chess board. Irregular = India ka political map (sab states alag size ke).

Quad Tree (4×4 raster example):

Raster: Quad Tree: F F I I Root → 4 quadrants F F I I ├─ Top-Left: all F → Leaf F R R F F ├─ Top-Right: all I → Leaf I R R F F ├─ Bot-Left: all R → Leaf R └─ Bot-Right: all F → Leaf F
Quad tree mein uniform regions = sirf 1 leaf node = storage efficient!
Geospatial Data
📡
Nov'19 Q1b · Nov'25 Q1d
Geospatial data, Geoinformation aur Metadata kya hain?
▼
  • Geospatial Data — Location se related raw facts: coordinates, satellite images, GPS readings
  • Geoinformation — Processed geospatial data jisme meaning ho: "Delhi mein kal baarish hogi"
  • Metadata — Data ke baare mein data: "Ye map 2020 mein banaya gaya, accuracy ±5m hai"

Spatial Data Quality ke key components:

  • Accuracy — Kitna sahi hai?
  • Completeness — Kuch chhoot toh nahi gaya?
  • Consistency — Data mein contradictions toh nahi?
  • Timeliness — Data kitna fresh hai?
